编程练习3：定义衡量标准Â¶

如果不能对模型的训练和测试的表现进行量化地评估，我们就很难衡量模型的好坏。通常我们会定义一些衡量标准，这些标准可以通过对某些误差或者拟合程度的计算来得到。在这个项目中，你将通过运算决定系数 R2 来量化模型的表现。模型的决定系数是回归分析中十分常用的统计信息，经常被当作衡量模型预测能力好坏的标准。

R2的数值范围从0至1，表示目标变量的预测值和实际值之间的相关程度平方的百分比。一个模型的R2 值为0还不如直接用平均值来预测效果好；而一个R2 值为1的模型则可以对目标变量进行完美的预测。从0至1之间的数值，则表示该模型中目标变量中有百分之多少能够用特征来解释。模型也可能出现负值的R2，这种情况下模型所做预测有时会比直接计算目标变量的平均值差很多。

在下方代码的 performance_metric 函数中，你要实现：

  • 使用 sklearn.metrics 中的 r2_score 来计算 y_true 和 y_predict的R2值，作为对其表现的评判。
  • 将他们的表现评分储存到score变量中。

或

  • (可选) 不使用任何外部库，参考决定系数的定义进行计算，这也可以帮助你更好的理解决定系数在什么情况下等于0或等于1。

from sklearn.metrics import r2_score
def performance_metric(y_true, y_predict):
    """计算并返回预测值相比于预测值的分数"""
    
    score = r2_score(y_true, y_predict)

    return score

def performance_metric2(y_true, y_predict):
    """计算并返回预测值相比于预测值的分数"""
    avg = sum(y_true)/len(y_true)
    tot = sum([(i-avg) ** 2  for i in y_true])
    res = sum([(i-j) ** 2  for i, j in zip(y_true, y_predict)])
    print "avg:", avg, ", tot:", tot, ", res:", res
    score = 1 - res/tot

    return score

  • GridSearchCV是如何结合交叉验证来完成对最佳参数组合的选择的？
  • GridSearchCV中的'cv_results_'属性能告诉我们什么？

提示： 在下面 fit_model函数最后加入 print pd.DataFrame(grid.cv_results_) 可以帮你查看更多信息。

问题 8 - 回答：Â¶

  • K折交叉验证法是一种对训练数据的处理方法；在给定的训练数据中分割得到更多的训练和测试数据的组合，相对于单一的对训练数据计算分数可以减少过拟合模型的计算分数。
    方法：在给定训练数据集上进行k等份分割，选取其中的1份作为测试集，其余的k-1做为训练集，按照指定的评分方法计算分数，依次进行k次后有k个得分，算出平均分做为该模型的分数。
  • 网格搜索交叉验证是输入算法，待测参数，评分方法，交叉验证方法，通过自动交叉验证每组参数组合，选取最佳参数组合的方法。
    与单纯的网格搜索法不同的是每组参数验证的时候采用交叉验证的方法。
  • cv_results_可以告诉我们：所有的参数组合的测试验证情况，每组参数组合的测试分数排名。
    其中每组参数组合都包括如下信息：参数组合ID，参数组合值(每个参数的取值)，交叉验证分组中每组验证的运行时间和分数（包括平均的训练集和测试集的运行时间和得分）
  • 不使用交叉验证进行网格搜索时，只对训练数据求分数，容易选出过拟合的模型。
    交叉验证将训练数据拆分成训练集和测试集，增加测试组合，减小该现象的发生概率。

from sklearn.model_selection import KFold
from sklearn.tree import DecisionTreeRegressor
from sklearn.metrics import make_scorer
from sklearn.model_selection import GridSearchCV

def fit_model(X, y):
    """ 基于输入数据 [X,y]，利于网格搜索找到最优的决策树模型"""
    
    cross_validator = KFold()
    
    regressor = DecisionTreeRegressor()

    params = {'max_depth':range(1,11)}

    scoring_fnc = make_scorer(performance_metric)

    grid = GridSearchCV(regressor, params, scoring=scoring_fnc, cv=cross_validator)

    # 基于输入数据 [X,y]，进行网格搜索
    grid = grid.fit(X, y)

    #print pd.DataFrame(grid.cv_results_) 
    
    # 返回网格搜索后的最优模型

from sklearn.tree import DecisionTreeRegressor

def div_list(X,y,n):   
    y_len = len(y) 
    #print "x:", len(X), ", y:", len(y)
    j = y_len/n  
    k = y_len%n  
    ### j,j,j,...(前面有n-1个j),j+k  
    #步长j,次数n-1  
    X_len = []
    y_len = []  
    for i in xrange(0,(n-1)*j,j):  
        #print "1-2:",i, i+j
        X_len.append(X[i:i+j])
        y_len.append(y[i:i+j]) 
        
    #算上末尾的j+k  
    X_len.append(X[(n-1)*j:])
    y_len.append(y[(n-1)*j:])  
    #print "3:", (n-1)*j, y_len
    return (X_len, y_len) 

def cross_val_score(estimator, X, y, scoring = performance_metric, cv=3):
    X_split, y_split = div_list(X_train, y_train,3)
    scores = []
    fit_x = X_split[1].append(X_split[2])
    fit_y = y_split[1].append(y_split[2])
    estimator.fit(fit_x, fit_y)
    pred = estimator.predict(X_split[0])
    scores.append(performance_metric(y_split[0], pred))
    
    fit_x = X_split[0].append(X_split[2])
    fit_y = y_split[0].append(y_split[2])
    estimator.fit(fit_x, fit_y)
    pred = estimator.predict(X_split[1])
    scores.append(performance_metric(y_split[1], pred))
    
    fit_x = X_split[1].append(X_split[0])
    fit_y = y_split[1].append(y_split[0])
    estimator.fit(fit_x, fit_y)
    pred = estimator.predict(X_split[2])
    scores.append(performance_metric(y_split[2], pred))
    
    #print "scores:" , scores
    avg_scores = sum(scores)/ len(scores)
    return avg_scores



def fit_model2(X, y):
    """ 基于输入数据 [X,y]，利于网格搜索找到最优的决策树模型"""
    
    scores = [cross_val_score(DecisionTreeRegressor(max_depth = i), X, y)  for i in range(1, 11)]
    max_index = scores.index(max(scores))  
    #最优交叉验证分数对应的最优模型
    best_estimator = DecisionTreeRegressor(max_depth = max_index+1)
    
    return best_estimator

def fit_model(X, y):
    """ 基于输入数据 [X,y]，利于网格搜索找到最优的决策树模型"""
    
    cross_validator = KFold()
    
    regressor = DecisionTreeRegressor()

    grid = grid.fit(X, y)

    return grid.best_estimator_
